# nexus-core

## 简易认证后端

### 用于个人订阅制软件，物联网设备等场景，提供一个简单的认证和监控后端。使用license Key认证等方式。

## 功能需求

### 登录？是否可以允许用户登录自助获取license
- 目前只提供了一个接口，管理员可以通过这个接口添加license key，用户可以通过这个接口获取license key。后续可以考虑添加用户登录功能
### 产品（软件产品，或物联设备？）
- 产品增加，修改，删除等
- 产品发布版本（即时发布，定时发布）
- 产品版本启用、废弃
- 产品启用、废弃
- 产品服务（调用服务实现功能如强制下线，可考虑向物联网类似扩展）
- 实现产品demo能够进行认证，通信测试
### 许可证
- 许可证增加，修改，删除等
- 许可证启用、废弃
- 许可范围设置，可许可多个产品
- 许可限制，绑定，并发限制
- 许可操作记录，绑定，激活解绑
### 节点（抽象的软件，物联设备，或一套用户运行环境）
- 节点通信协议http，mqtt？
- 节点心跳，动态创建
- 节点运行状态监控
- 节点绑定许可证，解绑许可证
- 节点运行环境信息
- 节点操作记录
- 节点黑名单机制
### 监控与统计
- 监控产品，许可证，节点的使用情况，异常情况等
- 监控在线与并发情况
