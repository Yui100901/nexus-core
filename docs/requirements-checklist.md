# nexus-core 需求推进清单

本文档用于跟踪 `nexus-core` 后续重构、功能实现和工程交付进度。

状态说明：

- `[ ]` 待处理。
- `[~]` 部分完成，需要继续补齐或整理。
- `[x]` 已完成。

优先级说明：

- `P0` 主链路能力，影响系统是否可用。
- `P1` 可验证性、文档和接口交付。
- `P2` 核心管理能力补齐。
- `P3` 监控、统计、审计和运营能力。
